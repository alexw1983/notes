Git Cheat Sheet

If you've already read the main bit here then this page summarizes that.

Clone

Note you only have to do this the first time. This will create a folder on your computer with all the code in it. You will need to be in that folder for the rest of this to work.

git clone path_to_repo

Create branch

Create a version of the code just for you.

git checkout -b branch_name

Publish branch

Let github know about your branch.

git push -u origin branch_name

Stage all changes

Preview yoru changes before saving.

git add .

Stage specific file

git add path_to_file

Commit Changes

Save your changes to git. Note this isn't the same as saving them on disk.

git commit -m "message"

Push Changes

Send your changes up to github.

git push

Pull Changes

Get the latest changes down from github

git pull
